## 使用 Node.js + Express + Bootstrap + MySQL 搭建博客系统

本教程将带你学会使用基于 Node.js 平台的 Express 框架及 Bootstrap 前端框架和 MySQL 来搭建一个博客系统。并通过该项目在实践中简单学习 Express、Bootstrap 和 MySQL 的使用。

学会搭建博客之后，开发其他网站也是大同小异。

各软件/框架版本：

+ Node.js  4.4.0+
+ Express  4.13.1+
+ Bootstrap  4.0 alpha
